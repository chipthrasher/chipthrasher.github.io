To submit a song to the Sunvox Music YouTube channel, please email stereofield2015@gmail.com with the following:

  1. Name that should be displayed on the song (e.g. NightRadio)
  2. Your song name (e.g. Pixel Cave)
  3. Track artwork to be displayed (please try to include a file larger than 700x700 px)
  4. Please also specify any links to your music pages that we should include.
